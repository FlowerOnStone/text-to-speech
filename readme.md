# Setup Texttospeech 

## Setup library 

### Google Text-To-Speech 
```
pip install gTTS 
```

### Django
```
pip install django 
```

### APScheduler
```
pip install apscheduler
```

## Create Database 
``` 
python manage.py makemigrations
python manage.py migrate
```

## Run server 
```
python manage.py runserver 
```
